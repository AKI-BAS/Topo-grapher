How to create site topography in ArchiCAD (Mesh from survey points)

To model terrain in ArchiCAD, generate an XYZ CSV of survey points from real elevation data and build a Mesh with the Mesh tool, or import a finished surface as DXF or IFC. All three come from the same site in Topo-grapher, so no GIS software is needed.

By Aki, founder of Topo-grapher · Updated June 2026

ArchiCAD's Mesh tool is built to turn 3D survey points into terrain, but getting accurate points usually means a surveyor or a GIS export. The shortcut is to generate the points from open elevation data and feed the Mesh tool directly. Here is the full workflow.

Step-by-step

  1. Define your site. Draw a polygon around the site on the map in Topo-grapher, or paste coordinates. The free tier covers up to 250m × 250m.
  2. Generate the points. Pick XYZ CSV for survey points to feed the Mesh tool, or DXF/IFC to import a finished surface.
  3. Bring the data in. Import the XYZ points, or use the DXF-DWG translator for a DXF terrain. IFC imports as a linked model element.
  4. Build the Mesh. Use the Mesh tool to generate a 3D terrain from the imported survey points; ArchiCAD triangulates them into a mesh surface.
  5. Set the site reference. Position the mesh against project zero so levels read correctly across the model.

Which format should you use?

XYZ CSV drives the native Mesh tool; DXF and IFC let you import a surface that's already built.

Frequently asked questions

Can ArchiCAD create a mesh from survey points?

Yes. The ArchiCAD Mesh tool builds a 3D terrain by triangulating imported 3D survey points. An XYZ CSV from Topo-grapher supplies those points, so you can generate site terrain without GIS software.

What format is best for ArchiCAD terrain?

Use XYZ CSV to drive the Mesh tool from survey points, or import a finished surface as DXF or IFC. Topo-grapher exports all three from the same site.

How accurate is the terrain?

It depends on the country's open data: 0.4m in Denmark, 1m across the UK, Sweden, Norway, France and parts of the US, 5-10m across most of Europe, and a 30m global fallback elsewhere.

Is there a free way to get site topography for ArchiCAD?

Yes. The free tier allows 3 generations per day for sites up to 250m × 250m at 12m spacing, with no account required.
